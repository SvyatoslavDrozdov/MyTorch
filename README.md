# MyTorch
Это учебный проект, в рамках которого я создаю аналог бибилотеки PyTorch, который позволяет использовать простые нейросети. На данный момент в этом проекте написаны основные классы, позволяющие создавать и использовать линейные и сверточные слои. Также реализованы ReLU, BatchNorm, Dropout. При этом используется готовая реализация torch тензоров для расчета градиентов.

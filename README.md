# MyTorch
Это учебный проект, а рамках которого я создаю аналог бибилотеки PyTorch, который позволяет создавать простые нейросети. На данный момент в этом проекте написаны основные классы, позволяющие создавать и использовать нейросети. При этом используется готовая реализация torch тензоров для расчета градиентов.
